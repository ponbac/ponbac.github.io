How to setup your session

Welcome to the NAO web interface!

To setup your session, simply enter the child's name and the NAO's IP-address and press submit. Your NAO will only use the child's name during the session, it will not saved for future sessions nor for any other purpose.

How to find the IP-address

The NAO's IP-address is easily accessible by simply pressing the button on it's chest.

Conducting a session

A session is conducted by progressing through the tasks that are found as tabs in the session window. Once a child is finished with a task, simply choose another task to continue the session.

It is recommended to initiate the session by running the 'Greeting' task to introduce the child to the robot. Likewise, running the 'Goodbye' task once the session is concluded provides a clear finish

Before the session

Make sure that the robot is fully charged before attempting to perform a session, the charger is best left disconnected during the session. Any other items needed for the session can be found in the specific tasks, review these to make sure everything is ready.

How to perform a task

Clicking the tab for a corresponding task opens the window in which the task is controlled. During the task the child is expected to provide different answers to questions asked by the NAO. The NAO itself can't interpret answers, so it's up to whoever is controlling the session to communicate with NAO what the child answered.

How to communicate with NAO

Whenever NAO expects an answer, a multitude of buttons are presented in the task-window. Press the button which best suits the child's response. This will continue the task appropriately.

Whenever the NAO is communicating with the child, a timer will appear in the task-window. This timer indicates how long till an answer is expected. Once the buttons appear, whoever is controlling the session has unlimited time in deciding which answer is the most appropriate. No timer is running in the background.
